# Counter App

A simple Counter Application built using React and Vite. This project demonstrates React state management using the `useState` hook.

## Features

* Increment counter value
* Decrement counter value
* Reset counter to zero
* Responsive and user-friendly interface
* Built with React and Vite

## Technologies Used

* React.js
* Vite
* JavaScript (ES6+)
* CSS

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/barnadeepmondal/Counter-App.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Counter-App/vite-project
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
